01 · Fondations

La philosophie qui change tout

Pas une copie de Netflix. Pas un Canal+ moins cher. Une plateforme construite autour d'une conviction : l'émotion est la prochaine interface.

01
Émotion First

Le streaming traditionnel classe par genre, format, popularité. Nous classons par état émotionnel. "J'ai besoin de me sentir vivant" est une intention valide. Notre moteur la comprend.

02
Contexte Intelligent

La plateforme s'adapte à qui vous êtes maintenant : heure de la journée, météo locale, rythme cardiaque si disponible, historique émotionnel. Chaque session est unique.

03
Social by Design

Regarder seul est une option, pas l'expérience par défaut. Rooms synchronisées, réactions en temps réel, commentary tracks partagés, clubs de visionnage asynchrones.

04
Créateur au Centre

Le contenu africain et francophone n'est pas un "segment". C'est le cœur. Des outils de distribution, analytics deep, revenus transparents pour chaque créateur sur la plateforme.

Netflix vous dit quoi regarder. Spotify vous dit quoi écouter. Notre plateforme vous demande comment vous vous sentez — et construit l'expérience à partir de là.

02 · Fonctionnalités

Ce que la plateforme fait vraiment

Chaque feature a une raison d'exister. Aucune n'est là pour cocher une case. Voici l'architecture fonctionnelle complète.

Couche
Expérience Spectateur
  • MoodEngine™ — Navigation par état émotionnel
    Au lieu de "Action", "Comédie" : "Je veux pleurer", "Je veux ressentir quelque chose", "J'ai besoin de m'évader". Un sélecteur d'émotion remplace le menu genre. L'IA mappe émotions → contenus via des embeddings sémantiques multi-dimensionnels.
    INÉDIT
  • WatchRoom — Co-visionnage synchronisé mondial
    Créez une room, invitez jusqu'à 50 personnes, regardez en parfaite synchronisation avec chat intégré, réactions flottantes, voice call optionnel. Fonctionne cross-device, cross-timezone avec compensation de latence.
    NEW
  • SmartResume — Reprise contextuelle intelligente
    Pas juste "reprendre là où vous en étiez". Si vous n'avez pas regardé depuis 3 semaines, la plateforme propose un "Previously on..." personnalisé — résumé IA des scènes clés. Si vous changez d'appareil, adaptation automatique de la qualité vidéo.
    IA
  • DeepFocus Mode — Streaming sans distraction
    Mode immersif qui désactive toutes les notifications système, assombrit les bords de l'écran (vignettage dynamique), synchronise les couleurs d'ambiance avec le contenu (ambilight software). Sur mobile : DND automatique, rotation bloquée.
    NEW
  • Commentary Track — Piste audio alternative
    Des créateurs, critiques, et fans peuvent enregistrer une piste audio alternative synchronisée. Regardez un film avec le réalisateur qui commente en direct. Ou un ami youtuber. Ou une version "vulgarisation culturelle" en langue locale.
    INÉDIT
  • ChapterMap — Navigation visuelle dans l'épisode
    Timeline enrichie avec miniatures, marqueurs d'intensité émotionnelle (pic de tension, moment clé, scène culte), marqueurs communautaires. On peut "jump to" la scène dont tout le monde parle sur les réseaux.
    NEW
  • LanguageLens — Adaptation linguistique intelligente
    Sous-titres adaptatifs : taille, police, couleur selon le contexte (action = gras, chuchotement = petit, humour = couleur). Traduction en wolof, haoussa, lingala, et autres langues africaines. Sous-titres phonétiques pour apprendre une langue en regardant.
    INÉDIT

    18h un vendredi ? Suggestion apéro-friendly. 23h seul un mardi ? Films introspectifs. Fête nationale ? Rail "Notre Histoire". Ramadan ? Mode adapté. La plateforme connaît votre contexte culturel et calendaire.

07 · Architecture

Le stack qui scale

Architecture cible pour supporter 1M d'utilisateurs sans réécriture majeure. Construire maintenant ce dont on a besoin dans 18 mois.

Apps
Next.js 15 + Expo

App Router, Server Components, streaming SSR. Expo pour iOS/Android/TV dans un seul codebase.

API
Fastify + tRPC

tRPC pour type-safety E2E. Fastify pour la performance pure. PostgreSQL + Redis pour la persistence et le cache.

Vidéo
Mux + Cloudflare Stream

Encoding multi-qualité automatique, DRM Widevine/FairPlay, CDN global avec PoPs africains, analytics vidéo intégrées.

IA / ML
OpenAI + Pinecone

Embeddings sémantiques pour MoodEngine, vector search pour recommandations, GPT-4 pour résumés "Previously On".

Realtime
Ably + WebRTC

WatchRoom synchronisation sub-100ms, réactions live, présence et curseurs temps réel dans les interfaces admin.

Paiement
Stripe + CinetPay

Stripe pour cartes internationales. CinetPay/Flutterwave pour Orange Money, Wave, MTN MoMo en Afrique francophone.

Infra
AWS + Terraform

Infrastructure as Code, multi-région, auto-scaling. Monitoring Datadog, alerting PagerDuty, SLA 99.9%.

Analytics
PostHog + Segment

Product analytics open-source self-hostable. Segment pour unifier les events. Metabase pour les dashboards créateurs et admin.
